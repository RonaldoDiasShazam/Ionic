NgIf

Condições

É utilizado sempre que você precisar alterar pedaços da pagina, conforme uma condição.

As condições só podem conter resultados/repostas true ou false.

Cada vez que clicar no botão ele inverte o valor da variavel administrador, que é booleana. Mudar true/false
Só aparece quando verdadeiro: Você é um Administrador

IF com ELSE

Para o ELSE é necessário criar um bloco html com a tag ng-template com o nome do bloco do ELSE.

Este código, irá receber uma idada no input e atualizará a varivel idade, exibindo se o usuário é maior ou menor de idade. Idade:
