Variáveis

{{ variavel }} - Exibe o que está dentro da variavel em tempo real. Isso se chama Interpolação.
Variável Tipo Valor
variavel {{variavel}}
qualquerCoisa any {{qualquerCoisa}}

texto string {{texto}}

Objetos

Para utilizar objetos devemos utilizar: varivel.propriedade
Variável Propriedade Valor
pokemon nome {{pokemon.nome}}
pokemon numero {{pokemon.numero}}
pokemon img {{pokemon.img}}

Modificando Variáveis em tempo Real (Binding)

Para modificar o conteudo de uma varivel em tempo real, precisamos de um input e nele conter: [(ngModel)]="nomeDaVar" Digite algo: Conteudo da Variável: {{variavel}}
