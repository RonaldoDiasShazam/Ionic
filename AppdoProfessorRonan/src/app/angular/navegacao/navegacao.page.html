Navegação

Navegação simples

Para navegar entre paginas de forma simples, basta adicionar a propriedade routerLink="/rota" em qualquer elemento da pagina. Navegar

Enviando dados simples

Para enviar dados de uma pagina para outra, enviando dados básicos, como um ID ou uma palavra, devemos fazer por programação, conforme os seguintes passos:
  • Criar uma pagina de destino: ionic generate page nomeDaPagina
  • Modificar ou criar uma nova rota no arquivo: app-routing.module.ts
  • No final do path colocamos o nome do parametro: nome-da-pagina:parametro
  • Criar o método responsavel por chamar a nova pagina, enviando o parametro.

Enviando dados complexos

Para enviar dados complexos ou muitos dados de uma pagina para outra, devemos fazer por programação, conforme os seguintes passos:
  • Criar uma pagina de destino: ionic generate page nomeDaPagina
  • Criar um serviço: ionic generate service servicos/nomeDoServico
  • Importar o serviço no arquivo: app.module.ts
  • Importar o serviço nas paginas.
